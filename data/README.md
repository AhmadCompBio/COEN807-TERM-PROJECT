Data Directory

Dataset Source
The dataset used in this project is from cars45.com, a Nigerian automotive marketplace.

Dataset Details
- Source: Kaggle - [Nigerian Car Prices Dataset] (https://www.kaggle.com/datasets/makindekayode/nigerian-car-prices-dataset)
- Size: 3,700+ vehicle records
- Features: Year, Mileage, Engine Size, Make, Model, Fuel Type, Transmission, Condition, Location
- Target Variable: Price (NGN)

Access Instructions

Option 1: Download from Kaggle
1. Go to the dataset link above
2. Click the "Download" button
3. Extract the ZIP file to get `car_prices.csv`
4. Place `car_prices.csv` in this directory (`data/`)

Option 2: Direct Download (Alternative)
- If the dataset is publicly available, you can also download it from:
- (https://drive.google.com/file/d/1UBRHewSV68xOm9N70R81zPxgLTBWvi8v/view?usp=sharing)
 
 Important Notes
- The raw dataset is not included in this repository due to file size constraints
- Please download it using the instructions above
- The dataset should be named `car_prices.csv` for the code to work correctly
