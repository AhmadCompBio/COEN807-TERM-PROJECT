Data Directory

Dataset Source
The dataset used in this project is from cars45.com, a Nigerian automotive marketplace.

Dataset Details
- Source: Kaggle - Nigerian Car Prices Dataset
- Size: 3,700+ vehicle records
- Features: Year, Mileage, Engine Size, Make, Model, Fuel Type, Transmission, Condition, Location
- Target Variable: Price (NGN)

Access Instructions
The dataset is available on Kaggle at:
https://www.kaggle.com/datasets/makindekayode/nigerian-car-prices-dataset

Important Note
The raw dataset is not included in this repository due to file size constraints.
Please download it from the source link above and place the `car_prices.csv` file in this directory.
