Results Directory

Contents
- `figures/` - All generated visualizations
  - `price_distribution.png` - Distribution of car prices
  - `correlation_matrix.png` - Feature correlation heatmap
  - `model_comparison.png` - Performance comparison
  - `actual_vs_predicted.png` - Actual vs predicted plots
  - `residual_analysis.png` - Residual analysis plots

- `performance_metrics.csv` - Summary of all model performances
- `feature_importance.csv` - Feature importance ranking

How to Reproduce
Run the `analysis.ipynb` notebook to generate all results.
